# AI Viral Clipper

AI-powered tool that automatically:
- extracts viral podcast moments
- generates vertical Shorts
- adds subtitles
- creates multiple clips
- formats for TikTok / YouTube Shorts

## Features

- AI transcript generation
- Viral hook detection
- Subtitle rendering
- Vertical 9:16 export
- Duplicate clip prevention
- GUI interface
- Auto clip generation

## Requirements

- Python
- FFmpeg
- Whisper AI

## Run

```bash
python gui.py
```

## Output

Generated clips appear inside:

```bash
clips/generated
```
